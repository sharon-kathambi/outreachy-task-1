# Task 1 - Wikipedia Article Formatter

Displays a formatted list of Wikipedia articles from a dataset.

## Description

Takes an array of Wikipedia page objects and prints them in the following format:

> Article "TITLE" (Page ID PAGEID) was created at MONTH DAY, YEAR.

## Technologies

- HTML
- Vanilla JavaScript
